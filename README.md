# MERN Google Auth

Sign In & Sign Up using google oauth and passport.js.

## Run Locally

Clone the project

```bash
git clone https://github.com/hammadhz/pms-client.git
```

Go to the project directory

```bash
cd pms-client
```

Write cmd to run server Locally

```
 npm start
```

## Tech Stack

**Client:** React, Tailwindcss
